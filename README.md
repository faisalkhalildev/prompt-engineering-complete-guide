# Prompt Engineering Complete Guide

This repository contains a GitHub-ready prompt engineering guide for developers who want more reliable results from AI tools in coding, debugging, planning, and code review workflows.

## Start Here

- Read the full guide: [Prompt_Engineering_Complete_Guide_v2.md](./Prompt_Engineering_Complete_Guide_v2.md)
- Original Word version: [Prompt_Engineering_Complete_Guide_v2 (1).docx](./Prompt_Engineering_Complete_Guide_v2%20(1).docx)

## What You’ll Learn

- How to write clearer coding prompts with role, stack, task, rules, and format
- How to debug with AI without wasting time on vague prompts
- How to use planning, prompt chaining, few-shot examples, and review prompts
- How to build a repeatable prompt library for real development work

- The guide has been cleaned for GitHub readability.
- Images are stored in the local [`assets/`](./assets/) folder instead of embedded base64 blobs.
